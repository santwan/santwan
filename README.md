## 🚀 Projects

### UPI Fraud Transaction Detection — Best Paper Award CIIR 2026
> *Final Year Project | Under mentorship of Prof. Anupam Ghosh, Ex-HOD CSE*

**Stack:** `Python` `TensorFlow` `Keras` `Scikit-learn` `Pandas` `NumPy`

- Trained and evaluated **3 RNN architectures** on 151K synthetic UPI transactions with **47:1 class imbalance** using a 20-timestep sliding window sequence input
- Conducted a **loss function ablation study** comparing BCE, Weighted BCE, and Focal Loss variants — identifying Focal Loss (γ=2.0, α=0.75) as the only stable configuration under severe imbalance
- Scaled SimpleRNN from 16→256 hidden units, empirically identifying a **performance ceiling at ROC-AUC ≈ 0.78** — proving vanishing gradients as the architectural bottleneck, not capacity
- LSTM(64) achieved **ROC-AUC 0.8368 and PR-AUC 0.4514** — outperforming SimpleRNN(128) by +7.0pp with only 7.3% more parameters and leading all five non-sequential baselines
